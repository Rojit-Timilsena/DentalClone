# Components Directory

This directory contains React components for the dental website.

## Implemented Components:

### Layout Components
- **Header.jsx** - Navigation and logo ✅
  - Responsive navigation with mobile toggle
  - Active section highlighting
  - Dropdown menu for Pages
  - Search modal integration
  - Accessibility support (ARIA labels, keyboard navigation)

- **SearchModal.jsx** - Full-screen search modal ✅
  - Full-screen overlay design
  - Keyboard support (Escape to close)
  - Auto-focus on search input
  - Prevents body scroll when open
  - Accessibility support

- **BackToTop.jsx** - Back to top button ✅
- **LoadingSpinner.jsx** - Loading spinner component ✅
- **DeveloperPopup.jsx** - Developer message popup ✅
